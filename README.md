[![Watch the video](https://img.youtube.com/vi/YOUTUBE_VIDEO_ID/maxresdefault.jpg)](https://www.youtube.com/watch?v=3oIYnT1b7z0) 

A lightweight Dalamud overlay that recreates the FNAF2 music-box winding minigame inside FFXIV.

## Quick Install
1. Add `https://slckcskn852.github.io/FNAFMusicBoxFFXIV/custom-repo.json` as a custom repo in Dalamud
2. Install "FNAF Music Box" from All Plugins
3. Toggle with `/musicbox`

## Quick Usage
- `/musicbox` — Toggle overlay
- Hold the wind button (click-and-hold) to charge the meter
- Right-click the button to open settings / pause

## Gamepad
- Alternating between L2 and R2 winds the music box!
- Recommended to increase the wind speed for controller usage.
- Take care of your hands!

## Features (short)
- Draggable, resizable overlay with circular meter and wind button
- Looping music, wind-up SFX, and fail-state audio/visuals
- Settings: opacity, scale, wind/decay speed, and volume

## Build
```bash
dotnet build "FNAFMusicBoxFFXIV.csproj"
```
